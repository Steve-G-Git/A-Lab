FEATURED PROJECT  ·  STUDY TOOL
CompTIA A+ Lab Terminal
A browser-based terminal simulator that responds to typed commands, parses input, and returns structured study content across all four CompTIA A+ domains. No libraries or frameworks — developed in vanilla JavaScript.
Input is captured via keyboard events, parsed against a command registry, and routed to the appropriate content or quiz handler. The quiz engine randomizes questions, evaluates typed answers against an accepted-answers array, and tracks XP state across the session.

DIAGNOSTIC TOOL  ·  LIVE
Network Troubleshooter
A scenario-based diagnostic tool structured around the CompTIA A+ troubleshooting methodology. Covers four scenarios: no internet access, WAN connectivity failure, broadcast storms, and APIPA addressing.
Each scenario walks through diagnosis step by step — identify the problem, establish a theory, test layer by layer, verify resolution, document findings. Correct and incorrect paths both return explanatory feedback so the tool teaches reasoning, not just answers.
